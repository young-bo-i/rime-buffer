系统要求

  • macOS 13 (Ventura) 或更高
  • Apple Silicon 或 Intel（通用二进制）

安装说明

  • 本安装器需要管理员密码，会把「Enter输入法」装到 /Library/Input Methods（所有用户可用）。
  • 应用为 ad-hoc 签名（未经 Apple 公证）。若首次打开安装器被拦，请在安装器上右键 → 打开，或在「系统设置 → 隐私与安全性」里允许。

安装之后

安装完成页会告诉你如何在「系统设置 → 键盘 → 输入法」里添加并启用它。全新安装的输入法通常需要注销并重新登录一次才会出现在列表中——这是 macOS 的正常行为。
